Mizan Finance Project

One project I have been working on is Mizan Finance Project. The goal of this project is to create a professional online space where investors can buy and sell stocks just like how they do on robinhood or coinbase. We have also added a AI tool where you can ask it questions regarding any stock, we have also included a discription of each stock to ensure every user can buy the stock just in case it goes against there religous beleif.

While building this website, I used HTML to organize the content and CSS to style the layout, navigation, images, and sections. I focused on making the website clean, easy to read, and consistent across all pages. This class and project helped me to do this and help out my friend whos the main creater behind this project to make sure all of the coding was done right.

This project is important because it shows my growth in web development and gives me a place to present my work in the future. I plan to continue improving the site by adding more links, options, and advanced AI settings.

Skills Used

  • HTML page structure
  • CSS styling
  • Navigation links
  • Website organization
  • Problem solving
  • Attention to detail

What I Learned

From this project, I learned how important it is to keep a website organized and consistent. I also learned how small design choices, such as spacing, colors, and navigation layout, can make a website easier to use. This project helped me become more confident with coding and gave me a stronger understanding of how websites are built.
