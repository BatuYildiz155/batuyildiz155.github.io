Portfolio Website Project

One project I have been working on is my personal portfolio website. The goal of this project is to create a professional online space where I can share my background, interests, skills, and academic progress as an Information Systems major at Hofstra University.

While building this website, I used HTML to organize the content and CSS to style the layout, navigation, images, and sections. I focused on making the website clean, easy to read, and consistent across all pages. This project helped me practice creating multiple connected web pages and using design choices that make the site look more professional.

This project is important because it shows my growth in web development and gives me a place to present my work in the future. I plan to continue improving the site by adding more projects, stronger visuals, and JavaScript features.

Skills Used

  • HTML page structure
  • CSS styling
  • Navigation links
  • Website organization
  • Problem solving
  • Attention to detail

What I Learned

From this project, I learned how important it is to keep a website organized and consistent. I also learned how small design choices, such as spacing, colors, and navigation layout, can make a website easier to use. This project helped me become more confident with coding and gave me a stronger understanding of how websites are built.
